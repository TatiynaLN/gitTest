подключение второго аккаунта git НЕ глобально

1 - команда git config user.name "ваше имя"

2 - команда git config user.email "ваш email"

Проверить на каком аккаунте

git config user.name

git config user.email

История коммитов

git log --oneline

Откат комита

  • Для начала – найдите коммит: (команда в истории комитов)
  • Теперь, используя его номер – выполните откат: git revert хеш комита(номер)
  • И загрузите изменения: git push origin Внимание все изменения вернуться к состоянию предыдущего комита
